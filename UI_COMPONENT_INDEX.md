# UI Component Index

This folder contains extracted React UI examples from the attached layout inspiration snippets. They are stored under the component library area and are not yet integrated into pages.

## Components

- GenieEffect — a polished dock-and-window concept with animated app cards and a glassy UI shell.
- SpotlightCard — a modular spotlight-style card layout for presenting charts and feature tiles.
- DirectionalDrawerDemo — a profile image editor surfaced from a slide-in drawer with motion.
- FileUploadDropzone — a compact upload surface with drag-and-drop style interaction.
- CreativeHoverCard — a hover-driven card set with layered images and animated transitions.
- AnimatedGridBackground — an animated SVG grid background that highlights cells on pointer movement.
- BackgroundHome — a simple wrapper for the 3D background scene example.
- BackgroundScene — a three.js scene composed with an orthographic camera and layered visual content.
- BackgroundModel — the underlying model and shader-driven mesh effect for the background scene.
